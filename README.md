# Todo App\n\nA simple, beautiful todo list application built with Next.js and Tailwind CSS.\n\n## Features\n\n- ✅ Add and delete tasks\n- ✅ Mark tasks as complete\n- ✅ Persistent storage (localStorage)\n- ✅ Task statistics\n- ✅ Responsive design\n- ✅ Zero database needed\n\n## Getting Started\n\n### Local Development\n\n```bash\nnpm install\nnpm run dev\n```\n\nOpen [http://localhost:3000](http://localhost:3000) in your browser.\n\n### Build for Production\n\n```bash\nnpm run build\nnpm start\n```\n\n## Deploy to Vercel\n\nThe easiest way to deploy is with [Vercel](https://vercel.com):\n\n1. Create a GitHub account and push this code to a GitHub repo\n2. Go to https://vercel.com and sign up with GitHub\n3. Import your GitHub repo\n4. Vercel will auto-detect Next.js and deploy\n5. Your app is live in seconds!\n\n## Tech Stack\n\n- **Framework:** Next.js 14\n- **Styling:** Tailwind CSS\n- **Storage:** localStorage (client-side)\n- **Hosting:** Vercel (free tier)\n\n## License\n\nMIT\n
